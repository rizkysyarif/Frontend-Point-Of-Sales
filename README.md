## Prerequisite

``` bash
- Node.js
- React.js
```

## Installation

### Clone
```
$ git clone https://github.com/rizkysyarif/Frontend-Point-Of-Sales
$ cd Frontend-Point-Of-Sales
```

### Install Depedencies
```
npm install
```

### Run server development
```
npm start
```

## Dependencies
```
- Axios
- Chartjs 2
- Moment
- Redux
- Reactstrap
- Rupiah Format
```
